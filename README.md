# Knight Moves

## Project Overview

Knight Moves is a Flutter-based chessboard application developed to demonstrate cross-platform mobile application development using Flutter and Dart. The application provides an interactive chessboard where users can select and move chess pieces using a simple tap-based interface.

This project primarily focuses on Flutter UI development, state management, responsive layouts, and navigation. It is intended as a learning project and currently does not implement complete chess rules or game logic.

---

## Features

- Interactive 8 × 8 chessboard
- Tap-to-select and tap-to-move functionality
- Selected square highlighting
- Responsive board layout
- Cross-platform support
  - Android
  - iOS
  - Web
  - Windows
  - Linux
  - macOS
- Clean and modular Flutter project structure

## Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/knight_moves.git
```

### Navigate to the Project

```bash
cd knight_moves
```

### Install Dependencies

```bash
flutter pub get
```

### Run the Application

```bash
flutter run
```

## Limitations

- The application currently does not enforce official chess rules.
- Turn-based gameplay has not been implemented.
- Check, checkmate, and castling are not supported.
- The project is intended to demonstrate Flutter development concepts rather than serve as a complete chess engine.
